# House Price Prediction using Linear Regression

## Task
Build a Machine Learning model to predict house prices based on features such as area, bedrooms, and bathrooms.

## Dataset
Housing Dataset containing:
- Area
- Bedrooms
- Bathrooms
- Price

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn

## Steps
1. Load dataset
2. Select features and target
3. Split dataset into training and testing sets
4. Train Linear Regression model
5. Predict house prices
6. Evaluate model using MSE and R² Score
7. Visualize Actual vs Predicted Prices

## Output

Example Results:

Mean Squared Error: 2750040479309.052

R² Score: 0.45592991188724463

Predicted Price: 4675650.79
